# Quiz for alphacore

## 使用說明

npm install

npm run dev

## 使用到的工具

1. Vite
2. React
3. React Router
4. TanStack Query
5. React Select
6. Tailwindcss

## 簡單說明

1. Vite 建立專案及安裝所需的 lib
2. 使用 React Router 建立 router，分別為 index, login 及 orders
3. login 會是一個由 dialog 組成的 Modal 元件
4. 使用 TanStack Query 來處理 api
5. 使用 react context 來管理
